# Data Layout

This release copy bundles the small MMLU / GPQA assets already present in the workspace:

- `mmlu-pro-redux`
- `supergpqa-redux`
- `supergpqa-redux-subfield`

It also bundles the lightweight BIRD metadata/query files under `data/bird`,
while omitting the much larger SQLite databases. See `data/bird/README.md` for
the expected layout and official download links for the databases.
